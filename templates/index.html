Usage tips

  • Search — filter by faction, keyword (INFANTRY / CHARACTER / VEHICLE …), T/W/points range, weapon name in any combination.
  • Army builder — pick a catalogue (faction data source), freely combine units from it, total points and size tiers update automatically.
  • My models — each model is bound to a datasheet; supports paint status, notes, multiple image uploads, model-type and per-slot loadout choices.
  • Browse — armies and model inventories of public accounts are visible to everyone; toggle public/private on your profile page.
