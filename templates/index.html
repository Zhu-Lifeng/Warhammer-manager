Usage tips

  • Search — filter by faction, keyword (INFANTRY / CHARACTER / VEHICLE …), T/W/points range, weapon name in any combination.
  • Army builder — pick a catalogue (faction data source), freely combine units from it, total points and size tiers update automatically.
  • Model lists — organise your painted/unpainted models into collections. Each list can be public (visible on this home page) or private. Customise its cover or let it cycle through your model photos.
  • Image editor — every uploaded image (model photo, list cover) can be cropped from its edit page.
